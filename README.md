## 🛠️ Setup Instructions

### 1. Get AdSense Account
1. Visit [Google AdSense](https://www.google.com/adsense/)
2. Sign up with your Google account
3. Add your website URL
4. Wait for approval (1-14 days)

### 2. Configure AdSense
Replace placeholder IDs in `index.html`:
```html
<!-- Replace YOUR_ADSENSE_ID with your actual client ID -->
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-YOUR_ADSENSE_ID"
        crossorigin="anonymous"></script>

<!-- Replace YOUR_AD_SLOT_ID with your actual ad unit ID -->
<ins class="adsbygoogle"
     data-ad-client="ca-pub-YOUR_ADSENSE_ID"
     data-ad-slot="YOUR_AD_SLOT_ID"></ins>
```

### 3. Deploy Website
- Upload files to your web hosting
- Ensure HTTPS is enabled (required for AdSense)
- Test on multiple devices
